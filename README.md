# 📚 Smart Study Planner

A simple and efficient web-based Study Planner application that helps students manage their daily tasks in an organized way.

-------------------------------------------------------------------------------------------------------
                                      _ ** Features**_
-------------------------------------------------------------------------------------------------------
 **CRUD Operations**
- Add new tasks  
- Edit existing tasks  
- Delete unwanted tasks  

 **LocalStorage Support**
- Tasks automatically saved in browser  
- Data remains even after page refresh  

 **Search Functionality**
- Quickly search tasks by name  

 **Priority System**
- Set task priority (High / Medium / Low)  

 **Toggle Complete**
- Mark tasks as completed or pending  

 **Task Counter**
- Shows total, completed, and pending tasks  

 **Responsive UI**
- Works perfectly on mobile, tablet, and desktop  

-------------------------------------------------------------------------------------------------------
## Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- LocalStorage API  

-------------------------------------------------------------------------------------------------------

##  Purpose

This project is built to improve productivity and help students manage study schedules in a simple and interactive way.
